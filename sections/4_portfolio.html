WINE QUALITY CLASSIFICATION

PYTHON

This is a Machine Learning project that predicts the quality of Red and White Wine out of (bad, good, or excellent) categories using these following features: sulfur dioxide, density, pH, sulphates, alcohol, and many more.

It was coded in a Jupyter Notebook environment and takes advantage of Libraries such as pandas, numpy, matplotlib, sklearn, ect. While the database came from an outside trusted source, I did have to do a lot analyzing with charts and preprocessing to get a better categorial numbers.

SNAKE EYES GUI

JAVA

This project demostrates java skills by creating a graphical user interface that allows the user/player to continuously roll a pair of dice until they both land on one (snake-eyes).

The score is accumulated by what you roll each throw and added to the overall. Through each roll the graphical pair of dice showcases what you got. This was coded on MobaXterm which is a known Unix environment.
